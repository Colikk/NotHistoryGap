# Security Policy

## Overview
This is a static HTML website with no backend, no database, 
and no user accounts. Potential vulnerabilities are limited 
in scope but reports are still welcome.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| All Versions | :white_check_mark: |
